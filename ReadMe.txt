GuruxDLMSLib Example contains Sample Client source codes.
Before you can read DLMS/COSEM meter you must set correct values.
We have set few example values in to the begin of main.

GuruxDLMSServer Example server creates four server. 

- Short Name DLMS Server to port 4060.
- Logical Name DLMS Server to port 4061.
- Short Name DLMS Server with IEC 62056-47 to port 4062.
- Logical Name DLMS Server with IEC 62056-47 to port 4063.

DLMS/COSEM is very complicated protocol. Do not hesitate to contact and ask.
